## 此文档不是什么“宝宝辅食”

这并不是一个全面讲解这些软件和模型如何使用的文档，例如此处并不会详细介绍软件和模型的使用方法。

也不要指望我会写什么能够马上让你用上的什么”XX一键YY脚本“。

## 此文档的作者不是你友好的免费客服专线

所以不要拿着你不知道从哪里弄来的第三方渠道/一键脚本/etc 来问作者你遇到的 XYZ 问题还不附上终端输出 ，谢谢合作。

## 参与贡献 / 讨论 / 回报问题
